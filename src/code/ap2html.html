ap2html

Per una facile gestione di siti web!

ap2html e' il programma che ho scritto per realizzare questo sito ed e' una riscrittura da 0 di ht2html, il programma con cui veniva generato il sito di python.org.

I risultati che ho ottenuto sono:

  1. Facile gestione di pagine scritte in diverse lingue (la vedete la bandierina a sinistra?);
  2. Facile inserimento di script pubblicitari e per il monitoraggio del sito (e.g. Adsense e Analytics);
  3. Realizzaione di una mappa di navigazione dipendente dalla pagina;
  4. Supporto per i fogli di stile (CSS);
  5. La pagina sorgente utilizza lo stesso stile della pagina finale;
  6. Altri dettagli...

Con una versione molto preliminare di questo programma ho realizzato pure il sito del Gruppo di Astrofisica delle alte energie dell'Università di Ferrara.

ap2html

Make site management easy!

ap2html is the software I developed to realize this and other websites. and it is rewritten from scratch after the inspiration of ht2html, the code that was used to generate the python.org website.

The results obtained so far:

  1. Easy management of pages written in several languages (look at the flag at the left...);
  2. Easy inclusion of ads and site monitoring scripts (e.g. Adsense e Analytics);
  3. Page dependent navigation map;
  4. Cascading Style Sheet (CSS) support;
  5. Source page uses the same CSS than the final one;
  6. Other details...

With a preliminar version of this software I also realized the website of the Ferrara University High Energies Astrophysics Group.
