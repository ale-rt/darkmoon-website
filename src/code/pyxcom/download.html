Installare pyxcom

Il codice sorgente di pyxcom si' puo' ottenere dai seguenti link (uno vale l'altro):

  • pyxcom-0.9.3.tar.bz2
  • pyxcom-0.9.3.tar.gz
  • pyxcom-0.9.3.zip

Una volta scompattato l'archivio troverete le istruzioni per installarlo ed usarlo.

Sono disponibili anche gli rpm per pyxcom:

  • pyxcom-0.9.3-1.noarch.rpm
  • pyxcom-0.9.3-1.src.rpm

Se l'installazione e' andata a buon fine potete avere una idea delle possibilita' di pyxcom con il comando:
pyxcom -h
Ovviamente in caso di problemi mandatemi, quesiti o richieste contattatemi e saro' ben felice di fare tutto il possibile per aiutarvi.
