Perche' e' nato?

Spesso mi e' capitato di dover utilizzare i dati dei coefficienti di attenuazione nei miei programmi di calcolo e analisi dati. Purtroppo XCOM nasce come un programma interattivo, dunque ho dovuto escogitare degli stratagemmi per poter utilizzare i suoi dati in maniera non interattiva. Alla fine mi sono ritrovato con una piccola libreria in Python che, una volta riorganizzata, divenne una interfaccia a riga di comando per XCOM.
Siccome l'appetito vien mangiando, ho cominciato ad aggiungere funzionalita' al programma, ad esempio un'interfaccia wxPython e l'ho pubblicato con il nome di pyxcom, sperando che qualcun altro traesse vantaggio di questo software libero come io lo traggo quotidianamente dal software libero pubblicato da altri. Dalla versione 0.9.2 ho pure aggiunto un'interfaccia PyQt.

Why pyxcom?

Several times it happened to me to use XCOM data in scripts. Unfortunately XCOM is an interactive program, so I had to found workarounds to comfortably access its results in scripts. After some coding I found I had a small Python library that, once reorganized, resulted in a TUI for XCOM.
Then I started adding features to this program, in particular I added an experimental wxPython interface and I released it to the pubblic domain under the name of pyxcom, hoping that some other people could take advantage of my work as I did with the one made by the Open Source community.
Since version 0.9.2 a PyQt GUI is also present.
