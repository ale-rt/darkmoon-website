Installare qombinatorics

Il codice sorgente di qombinatorics si' puo' ottenere dai seguenti link (uno vale l'altro):

  • qombinatorics-0.2.tar.bz2
  • qombinatorics-0.2.tar.gz
  • qombinatorics-0.2.zip

Una volta scompattato l'archivio troverete le istruzioni per installarlo ed usarlo.

Sono disponibili anche gli rpm per qombinatorics:

  • qombinatorics-0.2-1.noarch.rpm
  • qombinatorics-0.2-1.src.rpm

Gli rpm sono stati realizzati usando mandriva, ma dovrebbero funzionare su gran parte delle distribuzioni aggiornate. In ogni caso scaricando i sorgenti, troverete la descrizione per ottenere in maniera semplice rpm, deb o zip autoinstallanti per windows.
Ovviamente in caso di problemi mandatemi, quesiti o richieste contattatemi e saro' ben felice di fare tutto il possibile per aiutarvi.

Install qombinatorics

qombinatorics sources can be retrieved clicking on the following links:

  • qombinatorics-0.2.tar.bz2
  • qombinatorics-0.2.tar.gz
  • qombinatorics-0.2.zip

In the unpacked sources you will find the instruction to install and use, the software.

You can install qombinatorics also using prebuilt or sources rpm:

  • qombinatorics-0.2-1.noarch.rpm
  • qombinatorics-0.2-1.src.rpm

The rpms are realized using mandriva, but they should work fine on any recent distribution. You can also download the sources, there you will find instruction to build rpms, deb or self-installing zip for windows Of course if you find any problem in the installation phase, maybe the most critical one, please contact me and I will be really glad to help you.
