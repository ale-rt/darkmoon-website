pyxcom

pyxcom e' una interfaccia a riga di comando (TUI, Text User Interface) e grafica (GUI, Graphical User Inetrface) al programma XCOM rilasciato dal NIST (National Institute for Standards and Technologies).
pyxcom permette di calcolare in un baleno le sezioni d'urto e i coefficienti di attenuazione dei raggi X per parecchi materiali.

pyxcom

pyxcom is a pythonic GUI (Graphical User Inetrface) and TUI (Text User Interface) to the NIST (National Institute for Standards and Technologies) program XCOM.
pyxcom allows to calculate seemlessly the X-ray cross sections and attenuation factors of many materials.
