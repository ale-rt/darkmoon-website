ap2html

Make site management easy!

ap2html is the software I developed to realize this and other websites. and it is rewritten from scratch after the inspiration of ht2html, the code that was used to generate the python.org website.

The results obtained so far:

  1. Easy management of pages written in several languages (look at the flag at the left...);
  2. Easy inclusion of ads and site monitoring scripts (e.g. Adsense e Analytics);
  3. Page dependent navigation map;
  4. Cascading Style Sheet (CSS) support;
  5. Source page uses the same CSS than the final one;
  6. Other details...

With a preliminar version of this software I also realized the website of the Ferrara University High Energies Astrophysics Group.
